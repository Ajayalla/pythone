Please follow the below steps for lab exercises:

sudo docker build . -t user_name/myphythonapp
sudo docker images
the above command to see the image you had created
sudo docker login
enter your docker hub crendetials
sudo docker push user_name/myphythonapp

How to run application?

sudo docker run -p 8080:5000 --rm --name myfirstApp1 user_name/myphythonapp
now go to browser enter the public dns name with port no 8090.  

### if you would like to run on differet port 
sudo docker run -p 8090:5000 --rm --name myfirstApp2 user_name/myphythonapp
Make sure you open port 8090 in security firewall rules.
